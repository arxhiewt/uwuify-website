Important Setup Information

uwuify requires the following permissions to function correctly:

  • View Channels
  • Send Messages
  • Read Message History
  • Manage Messages
  • Manage Webhooks

Only users with any of the following roles can run uwuify commands:

  • Administrator
  • Ban Members
  • Kick Members
  • Timeout Members

Ensure your roles include the permissions above. This protects your server and ensures only trusted members can use the bot.

Commands

/uwuify <user>
/webhook <#channel>
/help

Privacy Policy

uwuify does not permanently store any user messages. User IDs and server IDs are temporarily stored for operational purposes. No user data is sold, shared, or distributed to third parties. By using uwuify, you consent to temporary processing necessary for bot functionality.
