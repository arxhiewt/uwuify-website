Documentation

/uwuify <user>

Transforms all messages of the mentioned user into uwu style for a set duration.

/webhook <#channel>

Sets the specified channel for live transformations via webhooks.

/help

Displays the full command list and usage instructions.

Important Setup Information

uwuify requires the following permissions to function correctly:

  • View Channels
  • Send Messages
  • Read Message History
  • Manage Messages
  • Manage Webhooks

Only users with any of the following roles can run uwuify commands:

  • Administrator
  • Ban Members
  • Kick Members
  • Timeout Members

Ensure any of your roles include the permissions above to run uwuify. This protects your server and ensures only trusted members can use the bot.

Privacy Policy

uwuify does not permanently store message content. User IDs and server IDs are temporarily stored for bot functionality. No data is sold or shared with third parties.
