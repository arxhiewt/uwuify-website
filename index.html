Important Setup Information

uwuify requires the following permissions to function correctly:

  • View Channels
  • Send Messages
  • Read Message History
  • Manage Messages
  • Manage Webhooks

Only users with any of the following roles can run uwuify commands:

  • Administrator
  • Ban Members
  • Kick Members
  • Timeout Members

Ensure your roles include the permissions above. This protects your server and ensures only trusted members can use the bot.

Privacy Policy

uwuify temporarily processes messages for live uwuification. Messages are deleted immediately after being sent. The only data stored temporarily is which users are currently uwufied; this data is removed automatically when the command ends or is manually removed. No permanent storage of messages, user IDs, or server information occurs.
